# fluid-api
A Spigot API designed to make plugin development a breeze

## How to use
```java
<repository>
  <id>jitpack.io</id>
  <url>https://jitpack.io</url>
</repository>
<dependency>
  <groupId>com.github.DefineDoddy</groupId>
  <artifactId>fluid-api</artifactId>
  <version>pre-0.1.0</version>
</dependency>
```
